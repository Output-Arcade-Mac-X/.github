## 📋 Overview

<a href="#outputArcade">Output Arcade</a> is a loop-based sample instrument that transforms audio loops into playable instruments — turning what would normally be background samples into pitched, manipulated, and remixed sound sources that respond to keyboard and MIDI input in musical ways.

The <a href="#outputArcade">pitched loop playback</a> is Arcade's core concept — loading an audio loop and playing it pitched across the keyboard, so pressing different keys plays the loop at different pitches while maintaining the rhythmic and textural character of the original sample. This turns a single drum loop into a pitched percussion instrument, a vocal chop into a melodic voice, or a guitar riff into a pitched texture that can be played harmonically. The musical relationship between loops and melody becomes directly playable from a keyboard rather than requiring complex pitch-shifting in a DAW. The <a href="#outputArcade">chop mode</a> divides loops into slices and assigns them across keyboard keys — enabling Arcade users to play loop segments like a traditional drum pad arrangement, triggering specific beats, syllables, or phrases from the loop in any order. Chop mode transforms a recorded groove into a playable instrument where the arrangement decisions happen in performance rather than in editing.

The <a href="#outputArcade">layer system</a> stacks up to four separate loops simultaneously — building layered textures from multiple sample sources playing together, with independent pitch, filter, and effect settings per layer. Stacked layers combine into a composite sound that plays as a single instrument, enabling the complex layered textures that modern music production uses without requiring multiple tracks and extensive routing. The <a href="#outputArcade">cloud sample library</a> provides continuously updated content through an Arcade subscription — new kits, loops, and sample packs added regularly so the available sound palette grows with each update rather than remaining static after the initial installation.

<a href="#outputArcade">Real-time time-stretching</a> adjusts the playback tempo of loops without affecting their pitch — syncing loops to the DAW project tempo automatically, stretching or compressing loop playback to fit the project's timing without the pitch artifacts that traditional time-stretching algorithms produce. <a href="#outputArcade">Filter and effects processing</a> shapes loop sounds with per-layer filter controls, reverb, delay, and other effects — modifying the character of each sample layer within Arcade before the signal reaches the DAW's effects chain.

<a href="#outputArcade">AUv3, AU, and VST plugin support</a> runs Arcade as a plugin within any compatible DAW on Mac — Logic Pro, Ableton Live, GarageBand, and other major DAW environments host Arcade as an instrument plugin rather than requiring the standalone application for production work. <a href="#outputArcade">MIDI control and automation</a> maps parameters to MIDI controllers for real-time performance control — assigning filter cutoff, layer volume, and effect parameters to hardware knobs and faders for expressive live performance and DAW automation.

Output Arcade's fundamental insight is treating loops as instruments rather than as fixed audio content. In most sample-based workflows, a loop plays back at its original pitch and tempo with limited interactivity — it's a recording that plays. Arcade makes loops playable by applying pitched playback across a keyboard, enabling the same loop content to be used melodically, harmonically, and rhythmically in ways that fixed playback doesn't support. This transforms the relationship between the producer and sample content from consumer to performer.

## 💬 FAQ

<details>
<summary>What makes Output Arcade different from a regular sampler?</summary>

Arcade treats loops as playable instruments — applying pitched playback across keyboard keys, chop mode for slice performance, and layer stacking for combined textures. Standard samplers typically play back samples at fixed pitch rather than making loops keyboard-playable as instruments.

</details>

<details>
<summary>Does Output Arcade work as a plugin in Logic and Ableton?</summary>

Yes. Arcade supports AUv3, AU, and VST plugin formats for use in Logic Pro, Ableton Live, GarageBand, and other compatible DAWs on Mac.

</details>

<details>
<summary>What is chop mode?</summary>

Chop mode divides a loop into slices and assigns them across keyboard keys — enabling performance of loop segments in any order as a playable instrument.

</details>

Up to four loops layer simultaneously with independent settings per layer.

</details>

</details>
